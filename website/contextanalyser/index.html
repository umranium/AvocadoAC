Context Analyser

Welcome to the Context Analyser. This application attempts to infer your current context by monitoring the device's sensors.

Your context consits of your current:

  • Activity - 'walking', 'sitting', etc,
  • Location - 'home', 'work', etc
  • Destination if you're travelling

The Context Analyser by itself doesn't let you see or do anything. This is it! You need to install one or more applications which integrate with the Context Analyser to see any benefit. Maybe you'd like to try one of these?

  • Places Displays places detected by the Context Analyser
  • Locale Plugin Allows you to program Locale conditions based on your actions
  • Context Home Screen A home screen which adapts to include context-aware suggestions
